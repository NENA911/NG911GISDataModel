# Flat-File GIS Data Model Template

This folder contains Python code capable of creating a NENA NG9-1-1 Data Model compliant template in both file geodatabase and GeoPackage (v3 or later) format. The folder is organized by release version and each release may support differing version of ArcGIS Desktop.

## Folders and Files

* [.imgs](.imgs) - Folder containing documentation images and screen shots.
* [v2.0a](v2.0a) - Folder containing code to create flat-file template to NENA-STA-006.2a specifications.
* [v3.0](v3.0) - Folder containing code to create flat-file template to NENA-STA-006.3 specifications.

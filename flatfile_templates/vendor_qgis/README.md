# QGIS Template Code

This folder currently serves as a placeholder for future QGIS template‑generation code. The objective to develop a custom Python plugin capable of creating a NENA NG9‑1‑1 GIS Data Model–compliant template in GeoPackage format.

We are seeking volunteers interested in contributing to the design and development of this plugin—particularly those with experience in QGIS plugin development, Python scripting, or familiarity with the NENA NG9‑1‑1 GIS data standards. Your help will accelerate the creation of an open, reusable tool for public safety GIS workflows
